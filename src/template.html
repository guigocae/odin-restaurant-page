About Page

This is a restaurant webpage I created as a training exercise to practice front-end development fundamentals.

Key Features & Approach:

  • Built with vanilla JavaScript, dynamically rendering all page elements.
  • Used Webpack for bundling, development workflow, and module management.
  • Structured the code using ES6 modules for better organization, inspired by component-based frameworks (like React) but without a virtual DOM.

While intentionally simple, this project helped me explore DOM manipulation, modular architecture, and build tools.
